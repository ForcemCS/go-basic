### 1. 指针 (Pointers) —— “门牌号” vs “房子本身”

#### 通俗解释

- **值传递 (Value)**：就像我有一个文档，你想要修改它，我**复印**了一份给你。你在复印件上乱涂乱画，但我手里的原件**没有任何变化**。
- **指针传递 (Pointer)**：我把这个文档存放的**柜子号码（内存地址）告诉你。你根据号码找到柜子，拿出文档修改。等你改完放回去，我再去拿时，看到的就是修改后**的内容。

#### 核心语法

- `&` (取地址)：获取变量的“门牌号”。
- `*` (解引用)：根据“门牌号”找到里面的东西。

```go
func upgradeUser(u *User) {
    u.Name = "VIP" 
}

func main() {
    me := User{Name: "小明"}
    upgradeUser(&me) 
    // 现在 me.Name 变成了 "VIP"
}
```

### 2. 流式处理 (Streaming) —— “喝自来水” vs “喝桶装水”

Go 从语言设计阶段，就把“流”当作一等公民，整个生态都是围绕它构建的

#### 通俗解释

- **非流式（Buffer/Load All）**：你想看一部 2小时的电影。必须先把 4GB 的文件**全部下载**到硬盘上，然后才能开始播放。如果你的硬盘只有 2GB，你就看不了了。
- **流式（Stream）**：像 Netflix 或 B站。**边下载边播放**。你只需要极小的内存缓存（比如几秒钟的画面），就可以看完几百 GB 的视频。

> **Go 的标准库几乎所有 IO 相关组件，都默认支持它们**

例如

| 场景        | 接口                             |
| ----------- | -------------------------------- |
| 文件        | `os.File` → `io.Reader / Writer` |
| HTTP 请求体 | `http.Request.Body`              |
| HTTP 响应   | `http.ResponseWriter`            |
| 网络连接    | `net.Conn`                       |
| 压缩        | `gzip.NewReader / Writer`        |
| 加密        | `crypto/*`                       |
| JSON        | `json.NewDecoder(r)`             |
| 日志        | `log.SetOutput(w)`               |
